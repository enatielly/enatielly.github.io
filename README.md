# Enatielly.github.io

Este repositório contém o código fonte da página pessoal de Enatielly Goes, hospedada no GitHub Pages.

## Sobre
A página foi desenvolvida para apresentar informações profissionais e acadêmicas, incluindo links para o currículo Lattes, Google Scholar, GitHub e ORCID.

## Tecnologias Utilizadas
- HTML
- CSS

## Como Visualizar
Acesse [https://enatielly.github.io](https://enatielly.github.io) para visualizar a página publicada.

## Estrutura do Repositório
- `index.html`: Arquivo principal da página.
- `style.css`: Arquivo de estilos.
