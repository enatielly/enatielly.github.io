## How to Edit

### 1. Clone the Repository
```bash
git clone https://github.com/enatielly/enatielly.github.io.git
cd enatielly.github.io
```

### 2. Edit Files
- `index.html` - Page content and structure
- `style.css` - Styling and layout
- `foto.webp` - Profile photo (replace with your own)

### 3. Commit and Push
```bash
git add .
git commit -m "Your descriptive message"
git push origin main
```

### 4. View Changes
The website will be automatically updated at https://enatielly.github.io within a few minutes.

## Common Tasks

### Add a Publication
1. Open `index.html`
2. Find the Publications section
3. Add a new `<div class="publication-item">` block
4. Commit and push

### Update Your Photo
1. Save your photo as `foto.webp` in the root folder
2. Commit and push

1. Open `index.html`
